# fiber-go-app

- go modules
- gofiber/fiber
- https://github.com/cosmtrek/air for live-reload

### dev

```bash
# to start hot reloading server
$ chmod +x air
$ ./air
```

### go modules

`go get _` to add new dependency
`go mod tidy` to prune deps in `go.mod`
`go mod vendor` copy installed deps to `/vendor`
